# Versioning Guidelines

Please follow the following [Semantic Versioning](https://semver.org/) release version
standard.

## Final Releases

    X.Y.Z

    Ex: 0.9.3
      2.9.1
      0.9.2

## Pre-releases

    X.Y.Z-alpha.N       # Alpha release
    X.Y.Z-beta.N        # Beta release
    X.Y.Z-rc.N          # Release Candidate

    Ex: 2.0.0-alpha.0
      2.1.0-rc.0
